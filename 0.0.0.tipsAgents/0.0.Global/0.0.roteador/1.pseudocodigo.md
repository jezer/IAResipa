
### Pseudo‑código do roteamento

```python
def route(user_prompt):
    intents = detect_intents(user_prompt)           # NLU leve (keywords + classifier)
    domains = detect_domains(user_prompt)           # taxonomia sua (negócio, arquitetura, dev, etc.)
    if len(domains) > 1:
        parts = split_prompt(user_prompt, domains)  # divide por domínio/assunto
        return [{"agent": choose_agent(d), "content": p} for d,p in parts]
    agent = choose_agent(domains[0])
    if not agent:
        return {"action":"reject",
                "message":"Nenhum agente cobre. Sugerir criação via AgenteCreator."}
    return {"agent": agent, "content": user_prompt}
```
